# Migration from previous versions

Create a separate Alb mode (`alb migrate https://alb.jmq.cz/2025-08-nurnberg/ local-dir-with-original-photos`)
which would harvest an album from a previous version and convert it to current one.

In `old/`, there are old Perl scripts generating those from `index.md`. You may expect that `index.md`
is indeed inside that local-dir-with-original-photos.
